Groovy

  • es un Lenguaje Dinámico para la Java Virtual Machine
  • es basado en Java pero con poderes adicionales
  • está inspirado en Python, Ruby y Smalltalk
  • tiene características de programación modernas
  • soporta DSLs
  • se integra con todas las clases y bibliotecas de Java
  • se compila directamente a bytecodes de Java
Características de Groovy

Groovy 1.8

Groovy 1.8 requiere JDK 1.7

Instalación de Java

  1. http://www.oracle.com/technetwork/java/javase/downloads/index.html
  2. Java SE Development Kit 7u21
  3. Crear variable de ambiente JAVA_HOME
  4. (Windows) Agregar %JAVA_HOME%\bin al PATH
  5. (GNU/Linux) Agregar $JAVA_HOME\bin al PATH

Instalación de Groovy

  1. http://groovy.codehaus.org/Download
  2. Groovy 1.8.9
  3. Descomprimir a C:\groovy-1.8.9
  4. Crear variable de ambiente GROOVY_HOME
  5. (Windows) Agregar %GROOVY_HOME%\bin al PATH
  6. (GNU/Linux) Agregar $GROOVY_HOME\bin al PATH

Programas instalados por Groovy

  • (Shell) groovysh
  • (GUI) groovyConsole
  • (Intérprete) groovy script.groovy

Bibliotecas externas

  • Agregarlas en %GROOVY_HOME%\lib

Primer programa Groovy



$ groovy -e "println 'Hello, world!'"
Hello, world!
